== Description ==

Help show support information online to your website, such as email, hotline and yahoo, skype status

== Installation ==

1. Download the plugin and extract its content
2. Upload the complete folder `support-online` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Appearance-> Widget and drang 'Fit-Support Online' to your sidebar
